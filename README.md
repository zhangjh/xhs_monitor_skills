# 📖 小红书舆情监控系列 (XHS Monitor Skills)

欢迎来到 OpenClaw 小红书舆情监控系列。本项目包含两个不同应用场景的技能包：

## 1. [🏮 小红书负面舆情监控 (xhs-monitor)](./xhs-monitor/SKILL.md)
*   **定位**：品牌危机公关。
*   **特点**：
    *   预置“品牌+负面词”渗透式扫描逻辑。
    *   **集成 Z总管 (AI Manager) 深度分析**。
    *   自动生成包含“核心风险洞察”和“策略建议”的高质量日报。

## 2. [🔍 小红书通用关键词监控 (xhs-generic-monitor)](./xhs-generic-monitor/SKILL.md)
*   **定位**：通用行业调研/话题追踪。
*   **特点**：
    *   无倾向性抓取，完全自定义搜索关键词。
    *   保留 7 天内最新笔记动态。
    *   纯净原始数据报告，适合作为数据源或轻量监控。
